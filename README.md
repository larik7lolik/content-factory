#  Content Factory — Контент-завод для вайб-маркетологов

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.12+](https://img.shields.io/badge/python-3.12+-blue.svg)](https://www.python.org/downloads/)

**Автоматическая генерация контента для вайб-маркетологов, продвигающих онлайн-школы и репетиторов иностранных языков**

---

## 🎯 Что это и зачем

Вайб-маркетолог в EdTech тратит **2-3 часа** на один пост для каждой площадки. Content Factory превращает **одну тему урока в готовый контент для 5 платформ с уникальной обложкой за 3-5 минут**.

> 💡 **Формула кейса:** Я сделал мультиагентный «Контент-завод» для вайб-маркетологов, которые продвигают онлайн-школы и репетиторов иностранных языков. Проект помогает решить проблему выгорания и рутины при создании ежедневного контента. Пользователь задаёт тему урока или тренд, а на выходе получает полностью упакованный пост для Telegram, VK, Дзена, блога и LinkedIn с уникальной тематической обложкой. Это полезно потому, что сокращает время создания одного поста с 2-3 часов до 5 минут, позволяя маркетологу держать высокий ритм публикаций без потери качества. Дальше проект можно развивать через подключение к базе знаний школы, анализ комментариев учеников для генерации тем и автоматическую публикацию в расписании.

---

## 🚀 Возможности

-  **Исследование темы** — автоматический сбор фактов и трендов из веба
- ️ **Кросс-платформенный контент** — адаптация под 5 площадок (Telegram, VK, Дзен, Блог, LinkedIn)
- 🎨 **Генерация обложек** — уникальные изображения через gpt-image-2
- 📤 **Автопубликация** — интеграция с Telegram, VK, Дзен через MCP-сервер
- 🎭 **Настройка тональности** — дерзкий, экспертный, поддерживающий стиль
- 📋 **JSON-контракт** — строгая типизация данных между агентами

### Субагенты

| Агент | Задача | Результат |
|-------|--------|-----------|
| **@researcher** | Сбор фактов из веба | `content/research.json` |
| **@content-writer** | Адаптация под 5 платформ | `content/output.json` |
| **@image-creator** | Генерация обложки | `content/cover.png` |
| **@publisher** | Публикация в соцсети | Пост в Telegram/VK |

## 📦 Быстрый старт

### Требования
- Python 3.12+
- OpenCode 1.16.2+
- API-ключи (OpenAI/ProxyAPI, Telegram Bot)

```bash
